Go 语言相等比较
2021-04-15 23:38   label Go

基本类型

  • 整型：int/uint/int8/uint8/int16/uint16/int32/uint32/int64/uint64/byte/rune/uintptr
  • 浮点型：float32/float64
  • 复数型：complex64/complex128
  • 字符串型：string
  • 布尔型：bool

比较规则：

  • 类型必须一致，否则不能比较。
  • 类型重定义不能直接比较，必须显式转换。
  • 类型别名可以直接比较。

派生类型

数组

  • 数组长度是类型的一部分，如果数组长度不同，无法比较。
  • 逐个比较元素类型和值。
  • 如果某个元素无法比较，则整个数组也无法比较。

struct

  • 逐个比较元素类型和值。
  • 如果某个元素无法比较，则整个 struct 也无法比较。

slice 和 map

  • 除了与零值 nil 比较，无法和其他任何值比较。
  • 使用工具函数比较，例如 bytes.Equal 和 reflect.DeepEqual。

函数

  • 除了与零值 nil 比较，无法和其他任何值比较。

channel 和 指针

  • 比较内存地址。

interface

  • 比较动态类型和动态值。
  • 如果动态类型不可比较，则会 panic。

小结

  • slice、map 和函数除了和零值 nil 比较，不能和其他任何值比较。
  • 派生类型任意组成元素不可比较，则整个类型不可比较。
  • channel 和指针比较的是内存地址。
  • 接口比较动态类型和动态值。
